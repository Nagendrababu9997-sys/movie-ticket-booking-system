MOVIE TICKET BOOKING SYSTEM

A simple Python project that allows users to book movie tickets and select food combos. This project is developed using Python Classes and Dictionaries.

FEATURES

Book movie tickets
Check ticket availability
Enter customer email
Select movie and number of tickets
Choose food combos
Calculate total amount automatically

TECHNOLOGIES USED

Python
Object-Oriented Programming (OOP)
Dictionaries
Conditional Statements

AVAILABLE MOVIES

movie1 → 40 tickets
movie2 → 10 tickets
movie3 → 0 tickets

AVAILABLE FOOD COMBOS

combo1 → ₹100
combo2 → ₹150

HOW IT WORKS

User enters email address.
User selects a movie.
User enters the number of tickets.
User chooses whether to add food combos.
Total amount is calculated and displayed.
Ticket availability is updated after booking.
